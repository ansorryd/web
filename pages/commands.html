Introduction

Ansorry is the perfect utility bot to spice up your server. The bot is awesome and can also be grown with your help.

About this Bot

The bot provides Moderation features with Games at it's core. The bot is hosted 24/7 and you can be reassured about it's uptime.

Technology

The bot is bulit with JavaScript and Python using the discord.js and py-cord libraries, same with the API the bot uses.

Ansorry'd uses it's own API to filter and provide high quality images to your server. With the help of our super fast and flexible API we are able to provide you the best and fastest features.

Getting Started

To get started with our bot is fairly easy and we have done our best to make it simpler for you.

Bot help

There is also a bot getting started command that you can use if you know the installation process.
You also have the option to get help on all the bot commands right through discord with the use of the bot's internal help command.

Important: Spamming and annoying people with the bot will not be tolerated, even the permission spam will be recorded.
Use n!report for more information.

Adding the bot

Commands

Syntax:
  • < > : Required Argument
  • [ ] : Optional Argument
  • ( ) : Command Aliases

General

  • Help
    • Description : Shows all the commands and how to use them.
    Usage : n!help

    Note : The commands help do not have aliases
  • afk
    • Description : Turns you into AFK mode and doesn't let others disturn you
    Usage : n!afk / n!afk off

    Note : When you turn afk off, there might be a duplicate message.
  • playstore
    • Description : Let's you search Android and iOS games
    Usage : n!playstore
    Example : n!playstore Discord
    Aliases: n!ps
    Note : Clicking on the link will take you directly to the Google PlayStore.
  • whois
    • Description : Shows you all the information of a user.
    Usage : n!whois

    Note : You cannot see any private information from this command.
  • serverinfo
    • Description : Shows information about a server
    Usage : n!serverinfo
    Aliases : None
